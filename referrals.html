Create new

Start from a blank viewer page before sharing a new sheet.

  1. Create a Google Sheet with columns named Member, Ideal Referral, Company, and Notes.
  2. Make the sheet public enough for anyone with the link to view it.
  3. Open the blank referral viewer.
  4. Paste the Google Sheet link into the blank page, load it, then use the copy link button to share that viewer.
The Google Sheet loader keeps the input side simple, so this page stays focused on browsing and sharing referrals.
